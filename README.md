공항 CCTV 안내방송
=============
공항 CCTV로 사람들의 혼잡도에 따른 안내방송

## 관련제품 (링크)
- [에스카](http://www.escacctv.com/bbs/board.php?bo_table=Product_01&wr_id=27)

## 한계점
공항의 사람 혼잡도에 따라서 사람들의 대기열이 지체되는 상황 발생

## 개선점
opencv와 yolo를 통하여 공항의 출입국 대기열 관리를 줄인다.
